Project description

This is fullstack web-application built using html,css,javascript for frontend, nodejs for backend and mongodb for database. User can create an account, for activating the user account email will be sent to user provided mail address after clicking the link user account will be created. After that user can login to the site post that user navigated to his/her personal dashboard their user can view the number of short urls created by him/her till date and can create short urls. User need to input long urls and web-application will generate short urls for the submitted long urls. User can use these short urls in place of long urls. If user forgot their password, he/she can click on the forgot password link, post that email will be sent to the user provided mail address and user can click on that link for changing the password.
