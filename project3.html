Project description

This is a front end web-application built using HTML,CSS and Javascript and it is a game in which there are three levels like easy,medium and hard. Time limit for each level is 1 minute based on the level the towers will be given for easy three towers,for medium four towers and for hard five towers and all these towers in each level will be placed one on the above on the source rod. The bigger one will be below and the smaller one will be above on the source rod, with in the time limit player should move all the towers from the source rod to destination rod using the middle rod and player cannot not place bigger tower on the top of smaller tower.
