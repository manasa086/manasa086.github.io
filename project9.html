Project description

This is a full-stack Customer Relation Management (CRM) Web-Application developed using html,css,react js for frontend , node js for backend and mongoDB for backend.Using this Web-Application Admin,Managerand Employee can create and view Leads,Service Request and Contacts.Employee can create an account . After creating an account, Email will be sent to the Employee Email Address for registration. After registering Employee will be allowed to Sign In to the Web-Application. When Employee Signs In she/he can view the lead details,contact details and service request. In order to Create new leads,service request,contacts Employee need to have access rights. Admin can provide access rights to Employee. Admin need to login in to the web-application using admin credentials.Post that,admin can change the status of employee and give access rights for employee for creation of leads,contacts,service request. Admin can make an employee as Manager. Admin can also Change the status of leads and service request. Manager can create new leads,contacts,service request and manager also allowed to change the status of leads and service request. When any user(Employee with accress rights,Admin,Manager) create new leads,service request and contacts mail will be sent to the Admin and Manager notifying that new lead/service request/contact create in the CRM Application.
