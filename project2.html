Project description

This is a front end web-application built using HTML,CSS and typescript. It is a tic tac toe game. First Player X/x (or) O/o will get the chance based on user input. After that in 3X3 grid each player will be given a chance to play the game. Any player who fill the grid with their selected symbol either diagonally,horizontally or vertically before the opponent will be declared as winner or game will be tie
