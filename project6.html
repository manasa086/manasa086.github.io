Project description

This is fullstack web-application built using html,css,reactjs for frontend, nodejs for backend and mongodb for database. User can login as admin to change the rights of employee. First Admin should give access rights to Employee to create invoices and admin has option to change the employee as manager.Manager can view all the invoices created by all the employee and manager can create new invoices. Employee can create an account. After getting the access rights from admin employee can login to the site post that employee navigated to his/her personal dashboard their employee can create new invoices , view created invoices and generate PDFs for the invoices.
