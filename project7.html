Project description

This is fullstack web-application built using html,css,reactjs for frontend, nodejs for backend and mongodb for database. User can login as admin to change the rights of seller. First Admin should give access rights to Seller to add products in to shopping List or reject the seller request.Seller can create Account.After getting an approval from admin,Seller can view all the products created by her/him. User can create an signup. Post that user need to verify his/her mail for logging in to Application .After SuccessFul login user will be navigated to his/her personal dashboard their user can add products to cart , view products in the site and buy the products .
