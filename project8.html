Project description

This is a fullstack amazon-clone web-application built using html,css,reactjs for frontend, nodejs for backend and mongodb for database. User can create account. Post creating the account user can login to the application. User can add items to the cart. After adding items to the cart user can buy the products by making the online payments. User can view the placed orders and user can cancel the placed orders. User can view the cancelled orders.
